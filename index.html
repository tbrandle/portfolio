MoneyBeetz

A pomodoro menubar app that allows users to start work time for 25 minutes with 5 minute breaks in between. User can choose how to spend the break by watching youtube videos, or make a beat with the built-in drum machine and random song generator.

Tech Stack: CSS3, Electron, React.js, Youtube API

MovieTracker

An app built using React, Redux, and Router4 to create an application that allows users to sign in/create an account, see a list of the most recent movies, and store their favorite movies.

Tech Stack: JavaScript, React.js, Redux, Router 4, Express, CSS, SQL, PostgresQL, Jest, Enzyme, MovieDB API

Breakout

The classic arcade game Breakout built using Canvas and Object Oriented JavaScript. The game has 4 levels, keeps track of score, and givers users 3 lives. UI is a 1950's theme.

Tech Stack: Object Oriented JavaScript, jQuery, Canvas, Mocha, Chai

TravelDen

A social media app focused on sharing travel itineraries. Users can sign in, create a travel itinerary, share the itineraries, favorite and like itineraries created by other users.

Tech Stack: React.js, Redux, Firebase, Firebase Authorization, Router 4, CSS, Jest, Google Maps API, Google Places API.

Weatherly

React.js app that allows users to specify a location and receive a weather forecast. App will render a ten day and seven hour forecast.

Tech Stack: React.js, WeatherUnderground API, CSS, WAI-ARIA, Webpack

2DoBox Pivot

Single page application to keep track of user's to-do list. User can rank list item by priority and filter the results. The app is fully accessabile to users with visual and auditory impairment.

Tech Stack: WEB-ARIA, Object Oriented JavaScript, jQuery, CSS, Selenium, Mocha, Chai
